### How to play

Collect more rings than your opponent.
Spin dash to knock their rings off!

### Controls

| Action        | Result  |
|:------------- | -----:|
| Left      | Move left |
| Right      | Move right |
| Up      | Move up |
| Down      | Move up NO I mean move down |
| Button | Spin dash |
